# echo

- `echo [text to print]`
- `echo 'text to print as it is'`
- `echo -e "interpret escape characters"`
- `echo "$var"` - prints the variable var

## Escape characters

- `\a` - bell
- `\b` - backspace
- `\c` - no newline
- `\f` - formfeed
- `\n` - new line
- `\r` - carriage return
- `\t` - tab
- `\v` - vertical tab
- `\\` - back slash
- `\0n` - ascii charcater represented by octal value n
