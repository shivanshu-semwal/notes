# Operating Systems

Every electronic device (which is somewhat complex) will have some operating system.
If you know the operating system better, you can customize it to your liking and
reduce time figuring stuff out.

## Storage - Filesystem

Every storage device you attach to your device will have some form of file system.
Some commonly used file systems are:

- FAT - [Wikipedia: File allocation table](https://en.wikipedia.org/wiki/File_Allocation_Table)
- NTFS - [Wikipedia: NTFS](https://en.wikipedia.org/wiki/NTFS)
- ESP - EFI system patition - [Wikipedia: EFI System Partition](https://en.wikipedia.org/wiki/EFI_system_partition)
- [Wikipedia: File System](https://en.wikipedia.org/wiki/File_system)
- [Wikipedia: Comparision of File Systems](https://en.wikipedia.org/wiki/Comparison_of_file_systems)
