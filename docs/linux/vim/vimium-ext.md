# vimium - a browser extension for vim users

- j up 
- k down
- u half page up
- d half page down
- shift + f - follow and open in new tab
- f - follow a link
- shift + h - forward
- shift + l - back
- shift j 
- shift k
- s close tab
- o - open bookmarks
- shift + t - search through open tabs
- ? - show shortcuts
