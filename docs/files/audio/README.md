# Audio Files

## Audio Codecs

- bitrate - the rate at which the instrument take sample

- .mp3
    - lossy
    - reduces bit rate
    - <https://en.wikipedia.org/wiki/ID3> - the metadata is stored in this format
- aac
    - lower bitrate
- ogg, vorbis
    - open source
- flac, alac
    - high quality
    - lossless compression
- .waf, aiff
    - no compression used
    - contain all information
    - for audio editing

- abi
- flv
- m4a
- mp3
- mpv
- mpeg1/2/4
- wmv
